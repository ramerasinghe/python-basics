# Pseudocode: Restaurant Ordering System

## 1. Start

## 1. Display Welcome Message
Display "Welcome to Python's Pancakes! We're happy to serve you."

## 1. Display Menu Items
Set menu = [
    ("Pancakes", 5.00)
    ("Omelette", 7.50)
    ("French Toast", 6.50)
    ("Coffee", 2.00)
    ("Orange Juice", 2.50)
]

Display "Checkout our menu:"
For each (item, price) in menu:
    Display item + "- $" + price
